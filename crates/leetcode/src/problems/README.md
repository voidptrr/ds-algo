# LeetCode Problems by Rating

Buckets:
- easy: rating < 1300
- medium: 1300-1699
- hard: >= 1700

Source: https://zerotrac.github.io/leetcode_problem_rating

If a rating is missing there, LeetCode difficulty is used first; otherwise a reasonable estimate is used.

Structure:
- easy/<problem_id>.rs
- medium/<problem_id>.rs
- hard/<problem_id>.rs
